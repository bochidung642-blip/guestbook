## Bước tiếp theo cần làm thủ công

**1. Cài Node.js** (chưa có trên máy): https://nodejs.org

**2. Sau khi cài, chạy:**
```bash
cd guestbook/web
npm install
```

**3. Setup `.env` cho backend:**
```bash
cd guestbook/api
cp .env.example .env
# Điền DATABASE_URL thực vào .env
```

**4. Chạy migration (khi đã có DB):**
```bash
cd guestbook/api
.venv/Scripts/alembic upgrade head
```

**5. Chạy dev:**
```bash
# Terminal 1 — backend
cd api && .venv/Scripts/uvicorn app.main:app --reload

# Terminal 2 — frontend
cd web && npm run dev
```

## Bước 4 — Deploy Backend (`/api`)

   - **Service Name**: `api`

   - **Service Name**: `web`
3. Vào tab **Variables**:
   - `NEXT_PUBLIC_API_URL` = URL của backend service (lấy từ tab **Settings** → **Domains** của service `api`)
4. Railway tự detect `package.json` → chạy `npm run build` rồi `npm start`
